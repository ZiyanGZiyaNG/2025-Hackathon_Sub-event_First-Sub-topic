# 彈幕射擊遊戲

這是一個使用 Python 和 Pygame 製作的簡單彈幕射擊遊戲。

## 安裝需求

1. 確保您已安裝 Python 3.x
2. 安裝必要的套件：
   ```
   pip install -r requirements.txt
   ```

## 如何遊玩

1. 運行遊戲：
   ```
   python game.py
   ```

2. 遊戲控制：
   - 使用左右方向鍵移動飛船
   - 按空白鍵發射子彈
   - 擊中敵人獲得 10 分
   - 被敵人撞到會減少生命值
   - 生命值歸零遊戲結束

## 遊戲特點

- 簡單的射擊機制
- 分數系統
- 生命值系統
- 隨機移動的敵人
